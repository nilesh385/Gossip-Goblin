# Real-time Chat Application

This project is a real-time chat application built with React, TypeScript, and Node.js. Key features include user authentication, one-to-one messaging, friend requests, and a user-friendly interface.

## Features

- **Real-time Messaging:** Experience instant communication with other users.
- **One-to-one Messaging:** Send direct messages to individual users.
- **Friend Requests:** Connect with other users by sending and accepting friend requests.
- **User Authentication:** Securely manage user accounts and logins.
- **User-Friendly Interface:** Enjoy a clean and intuitive user experience.

## Technologies Used

- **Frontend:** React, TypeScript
- **Backend:** Node.js, Express.js, Socket.IO
- **Database:** MOngoDB
